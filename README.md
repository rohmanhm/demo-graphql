# Simple Djavaweb GraphQL
> Simple Djavaweb GraphQL Implementation

## Prerequisite
- NodeJS

## How to run?
1. Clone this project to your local directory
2. Install all dependencies
    ```bash
    npm install
    ```
3. Build and start project by
    ```bash
    npm run start
    ```
4. Open in your browser
    ```bash
    http://localhost:3000/
    ```
5. DONE.
